# USM CS Year 2 - Online Messaging App (Spring Boot Project)
Online web application for messeging 

General structure:

User          - date cont + autentificare

Chat          - conversatie, are o lista de User

Message       - mesaj, apartine unui Chat si unui User

UserSettings  - setarile unui user (tema, notificari etc.)

AppConfig     - info despre aplicatie (versiune, descriere)
